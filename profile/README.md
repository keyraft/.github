# Keyraft

**Open-source configuration and secrets management.**

Keyraft is a GitHub organization building tools to store, version, and distribute application configuration and secrets - self-hosted, simple to deploy, and easy to integrate from any language.

---

## What we build

We maintain a small, focused ecosystem:

- **Server** - a single-binary backend with HTTP/JSON API, encryption, RBAC, audit logs, and a web UI
- **Client SDKs** - official libraries so apps can read and write config without hand-rolling HTTP calls
- **Shared standards** - consistent API design, auth model, and licensing across all repos

---

## Philosophy

- **Self-hosted first** - your data stays on your infrastructure
- **Boring tech** - HTTP, JSON, tokens; no complex platform lock-in
- **Small surface area** - one server, clear API, thin SDKs
- **Open by default** - Apache 2.0 across public projects
