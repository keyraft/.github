# Keyraft

Lightweight, self-hosted configuration and secrets management system.

## About

Keyraft provides a simple, secure way to manage configuration and secrets for your applications. Built with Go, it offers a single binary deployment with no external dependencies.

## Projects

### [keyrafted](https://github.com/keyraft/keyrafted)
Core server implementation. Single-node configuration and secrets management with BoltDB storage, AES-256-GCM encryption, and real-time watch API.
